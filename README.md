# Vulnerability Assessment Report

## Target
testphp.vulnweb.com

## Tools Used
- Nmap
- OWASP ZAP
- Browser DevTools

## Scope
Passive security testing of publicly accessible demo website.

## Key Findings
- No HTTPS (High)
- Outdated PHP version (High)
- Server version disclosure (Medium)
- Insecure cookie configuration (High)
- Missing security headers (Medium)

## Overall Risk Rating
HIGH

## Disclaimer
This assessment was conducted for educational purposes on an intentionally vulnerable testing website.
